Whatsapp decrypt messages DB
======

Bash script to decrypt messages Whatsapp on android
need to install openssl and sqlite3

Linux and Mac already
-----------------------


How to run:
======

sh decrypt.sh "filename.db"
------------------------

Example:

sh decrypt.sh msgstore-2013-04-16.1.db
